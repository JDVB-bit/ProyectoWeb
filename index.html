¿Qué es una Página Web?

Una página web es un documento digital que contiene información en forma de texto, imágenes, videos y enlaces. Está almacenada en una computadora especial llamada servidor web, que está encendida las 24 horas del día para que cualquier persona en el mundo pueda acceder a ella en cualquier momento.

Para ver una página web, el usuario utiliza un programa llamado navegador de internet (como Google Chrome, Mozilla Firefox, Microsoft Edge o Safari). El navegador envía una solicitud al servidor usando una dirección única llamada URL (Uniform Resource Locator), el servidor responde enviando los archivos de la página, y el navegador los convierte en la interfaz visual que vemos en pantalla.

Toda esta comunicación entre el navegador y el servidor ocurre mediante el protocolo HTTP (HyperText Transfer Protocol). Cuando la dirección empieza con HTTPS, significa que la conexión está cifrada y protegida, lo cual es especialmente importante en sitios de bancos, tiendas online o redes sociales donde se manejan datos personales.

💡 Dato importante: Una página web es un solo documento (como una hoja de papel), mientras que un sitio web es el conjunto completo de todas las páginas bajo un mismo dominio. Por ejemplo, la página de inicio de Wikipedia, un artículo sobre Colombia y la página de búsqueda son tres páginas diferentes, pero todas forman parte del mismo sitio web: wikipedia.org.

Historia de la Web

La World Wide Web (WWW) fue inventada en 1989 por el científico británico Tim Berners-Lee mientras trabajaba en el laboratorio de física CERN en Suiza. Su idea original era simple pero revolucionaria: crear un sistema para que los científicos de todo el mundo pudieran compartir documentos e información fácilmente a través de internet. La primera página web de la historia se publicó el 6 de agosto de 1991 y aún puede visitarse en info.cern.ch.

Desde entonces, la web ha evolucionado enormemente. Los expertos la dividen en tres grandes etapas:

  • Web 1.0 — La web de solo lectura (años 90): Las páginas eran completamente estáticas, como revistas digitales. El usuario solo podía leer el contenido, no podía comentar, publicar ni interactuar. Los sitios eran creados exclusivamente por empresas o instituciones con conocimientos técnicos avanzados.
  • Web 2.0 — La web social e interactiva (2004 en adelante): Gracias al avance de JavaScript y las bases de datos dinámicas, nacieron plataformas donde los usuarios podían crear y compartir su propio contenido. Surgieron redes sociales como Facebook (2004), YouTube (2005) y Twitter (2006), además de Wikipedia y los blogs masivos. El usuario pasó de ser espectador a ser creador.
  • Web 3.0 — La web inteligente y descentralizada (actualidad): La web actual usa Inteligencia Artificial para personalizar la experiencia de cada usuario. Los buscadores ya no solo buscan palabras clave, sino que entienden el contexto y la intención del usuario. También surge la descentralización mediante tecnologías como el blockchain y los NFTs.

Estructura de una Página Web

Toda página web moderna se construye combinando tres lenguajes que trabajan juntos de forma coordinada. Cada uno tiene un rol completamente diferente y ninguno puede reemplazar al otro:

🏗️ HTML

HyperText Markup Language. Es el esqueleto de la página. No es un lenguaje de programación, sino un sistema de etiquetas que define el contenido: títulos, párrafos, imágenes, listas, tablas y formularios. Sin HTML no existe ninguna página web. Versión actual: HTML5.

🎨 CSS

Cascading Style Sheets. Es la ropa y el maquillaje de la página. Controla todos los aspectos visuales: colores, fuentes, tamaños, espaciados, animaciones y el diseño responsivo, que hace que el sitio se vea bien tanto en computadores como en celulares y tablets. Versión actual: CSS3.

⚡ JavaScript

El único lenguaje de programación nativo de los navegadores. Añade vida e interactividad: menús desplegables, validaciones de formularios, animaciones, carga de contenido sin recargar la página (AJAX) y mucho más. Frameworks populares basados en JS: React, Vue y Angular.

🧠 Analogía: Si una página web fuera una persona, el HTML serían los huesos (estructura), el CSS sería la ropa y apariencia (estética), y el JavaScript serían los músculos y el cerebro (movimiento e inteligencia).

Componentes de una Página Web

Con el estándar HTML5, las páginas web se organizan usando etiquetas semánticas: etiquetas cuyo nombre describe claramente el tipo de contenido que contienen. Esto ayuda a los navegadores a entender la estructura de la página, mejora la accesibilidad para personas con discapacidades visuales (que usan lectores de pantalla), y ayuda a los buscadores como Google a indexar el contenido correctamente para el SEO.

Una página web bien construida debería tener todos estos componentes:

Componente Etiqueta HTML5 ¿Para qué sirve?
Encabezado <header> Va en la parte superior de la página. Contiene el logo, el nombre del sitio y el menú de navegación principal.
Navegación <nav> Agrupa los enlaces del menú que permiten moverse entre las secciones o páginas del sitio. Es fundamental para la accesibilidad y el SEO.
Contenido principal <main> El corazón de la página. Contiene el contenido más importante y único del documento. Solo puede existir uno por página.
Sección <section> Divide el contenido principal en bloques temáticos. Cada sección debe tener su propio título (h2, h3...).
Artículo <article> Contiene información independiente que podría publicarse por separado: una noticia, entrada de blog, tarjeta de producto o comentario.
Barra lateral <aside> Contenido secundario o complementario ubicado al costado del contenido principal. Se usa para menús adicionales, glosarios, publicidad o widgets.
Pie de página <footer> Va al final del documento. Contiene información legal, derechos de autor (copyright), políticas de privacidad, redes sociales y datos de contacto.
📌 ¿Por qué usar etiquetas semánticas? Antes de HTML5, los desarrolladores usaban etiquetas genéricas como <div> para todo. El problema es que un <div> no le dice nada al navegador ni a Google sobre qué tipo de contenido tiene. Con etiquetas como <header>, <nav> o <footer>, tanto las máquinas como los humanos entienden inmediatamente de qué trata cada parte de la página.

📖 Glosario

URL: Dirección única de una página en internet (ej: https://google.com).

Servidor: Computadora siempre encendida que almacena y entrega las páginas web a quien las pida.

Dominio: El nombre que identifica un sitio web (ej: google.com, youtube.com).

HTTPS: Protocolo seguro de transferencia de datos. Cifra la información para proteger al usuario.

SEO: Search Engine Optimization. Técnicas para que una página aparezca en los primeros resultados de Google.

Hosting: Servicio que alquila espacio en un servidor para guardar los archivos de un sitio web.

DNS: Sistema que traduce nombres de dominio (google.com) a direcciones IP numéricas que los servidores entienden.

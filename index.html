¿Qué es una Página Web?

Una página web es un documento digital que contiene información en forma de texto, imágenes, videos y enlaces. Está almacenada en un servidor y se accede a ella desde un navegador (Chrome, Firefox, Safari) usando una dirección única llamada URL.

El navegador recibe los archivos del servidor y los traduce en una interfaz visual que el usuario puede ver e interactuar. La comunicación entre el navegador y el servidor se hace mediante el protocolo HTTP o HTTPS (la "S" significa que la conexión es segura y cifrada).

💡 Dato: Una página web es un solo documento, mientras que un sitio web es el conjunto de varias páginas bajo un mismo dominio (por ejemplo, todas las páginas de wikipedia.org forman un sitio web).

Historia de la Web

La World Wide Web fue creada en 1989 por el científico británico Tim Berners-Lee en el laboratorio CERN de Suiza, con el objetivo de compartir información científica entre investigadores de todo el mundo.

  • Web 1.0 (años 90): Páginas estáticas y de solo lectura. El usuario solo podía ver el contenido, no interactuar.
  • Web 2.0 (2004 en adelante): Nacen las redes sociales, los blogs y el contenido creado por el usuario (YouTube, Facebook, Wikipedia).
  • Web 3.0 (actualidad): Inteligencia artificial, personalización y descentralización. Los buscadores entienden el contexto, no solo las palabras.

Estructura de una Página Web

Toda página web moderna se construye con tres lenguajes que trabajan juntos:

🏗️ HTML

Define el contenido y la estructura: títulos, párrafos, imágenes, listas y formularios. Es el esqueleto de la página.

🎨 CSS

Controla la apariencia visual: colores, tipografía, tamaños y la adaptación a distintos dispositivos (responsivo).

⚡ JavaScript

Añade interactividad: animaciones, validaciones de formularios, menús desplegables y carga de datos sin recargar la página.

Componentes de una Página Web

Con el estándar HTML5, las páginas se organizan con etiquetas semánticas que le dicen tanto al navegador como a Google qué tipo de contenido contiene cada parte:

Componente Etiqueta ¿Para qué sirve?
Encabezado <header> Logo, nombre del sitio y menú de navegación.
Navegación <nav> Los enlaces que permiten moverse por el sitio.
Contenido principal <main> El corazón de la página, el contenido más importante.
Sección <section> Divide el contenido en bloques temáticos.
Artículo <article> Contenido independiente como una noticia o entrada de blog.
Barra lateral <aside> Información complementaria al costado del contenido principal.
Pie de página <footer> Datos de contacto, copyright y enlaces secundarios.

📖 Glosario

URL: Dirección única de una página en internet.

Servidor: Computadora que almacena y entrega las páginas web.

Dominio: Nombre del sitio (ej: google.com).

HTTPS: Protocolo seguro de transferencia de datos.

SEO: Técnicas para aparecer primero en los buscadores.
